Chapter 4 Terms

Operating Systems Terminology
TERM DEFINITION
Network Classifications
  • LAN: consists of a collection of computers in a single building or building complex
  • MAN: a network of intermediate size, such as one spanning a local community
  • WAN: links machines over a greater distanceâ€”perhaps in neighboring cities or on opposite sides of the world
Network Models
  • bus network: A network created by connecting all machines to a central communication line known as the bus.
  • star network: A network in which all machines are connected to a central machine.
Protocols
  • CSMA/CD Ethernet: "Carrier Sense, Multiple Access with Collision Detection" dictates that each message be broadcast to all machines on the bus but only the intended recipient keeps the message. A machine waits for the bus to go silent before transmitting a message and if a collision is detected both machines go silent for a random period of time before trying again.
  • CSMA/CA Wifi: "Carrier Sense, Multiple Access with Collision Avoidance" Prioritizes avoiding collisions as opposed to detection. The most common approach is to give advantage to machines that have already been waiting. The machine only transmits its message after a brief pause in which it waits to see if the channel remains silent. If the channel is busy, the machine waits for a random amount of time before trying again.
Combining Networks
  • repeater: a device that passes signals back and forth between the two original buses (usually with some form of amplification) without considering the meaning of the signals
  • bridge: a device that connects two buses but does not pass all messages, instead, it looks at the destination address and forwards a message across the connection only when when that message is destine for a computer on the other side.
  • switch: this device is essentially a bridge with multiple connections to buses.
  • router: special purpose computers used for forwarding messages between networks to form an internet.
  • gateway: the point where a network is linked to the internet, this is useually the router.
Client/Server Model
  • P2P (peer-to-peer): a model that involves processes that provide service to and recieve service from eachother
  • distributed system: a system that consist of softwre units that execute as processes on different computers
  • coud computing: a system in which huge pools of shared computers on the network can be allocated for use by clients as needed
Internet Addressing
  • Tier 1: consist of a few high-speed, high-capacity international WANs. They are the "backbone" of the internet.
  • Tier 2: smaller (regional) scope than Tier 1, they connect to Tier 1 ISPs.
  • Tier 3: provides access to the core of the internet (Tier 1 and Tier 2 routers). It is sometimes called an intranet, is operated by a single authority that is in the business of supplying internet access to individual homes and businesses.
IP Address
  • ICANN: "Internet Corporation for Assigned Names and Numbers" is a non-profit corporation who is tasked with coordinating the internet's operation. One such task is to award consecutively numbered IP addresses to ISPs.
  • DNS: "Domain Name System" a collection of name servers which convert mnemonic names into IP addresses.
  • TLD: "Top-level domain" suffixes added to the domain name which refelct their classification (e.g. "".edu")
  • Country-Code TLDs: TLDs which indicate a specific country
Internet Protocols
  • FTP: "File Transfer Protocol" a protocol used for transferring files from a server to a client over a network.
  • HTTP: "Hyper Text Transfer Protocol" is a protocol for distributed, collaborative, hypermedia information systems. It forms the foundation of data communication on the World Wide Web.
  • SMTP: "Simple Mail Transfer Protocol" defines a way that two computers on the network may interact when transmitting an email from one to the other.
World Wide Web
  • URL: "Uniform Resource Locator" a unique address which contains the information needed by a browser to contact the proper server and request the desired document.
  • client-side activities: activities performed by the client in regards to a webpage.
  • server-side activities: activities performed by the server in regards to a webpage.
Internet Four-Layer Software
  • application layer: the layer in which the message usually originates
  • transport layer: accepts messages from the application layer and ensures that the messages are properly formatted for transmission by dividing long messages into smaller segments which are transferred sequentially. This layer adds sequences of numbers to the smaller segments so that they can be reassembled on the other side and then transfers these packets to the network layer.
  • network layer: This layer is tasked with deciding which direction a packet should be sent at each step along the packet's path through the Internet.
  • link layer: This layer is tasked with transferring the packet. This layer deals with the communication details particular to the individual network. For example it applies a different protocol based on if the computer is apart of a WiFi or Ethernet network.
Internet is:
  • packet-switching network: digital data is sent by breaking the data into blocks of bits called packets, which contain both the data being transmitted and control information for routing the data
  • fault tolerant: The internet is able to operate even if a critical ISP goes offline
  • redundant: There is more than one pathway to route data between to points which increases the reliability of the internet and helps it scale to more devices and people.
Cybersecurity
  • malware: malicious software including viruses, worms, Trojan horses, and spyware.
  • virus: software that infects a computer by inserting itself into programs that are already on the machine. When the "host" program is run it initiates the execution of the virus also. The virus usually transfer themselves to other programs but can perform more nefarious actions such as degrading the computer in various manners.
  • worm: an autonomous program that transfers itself through a network and takes up residence in computers and forwards copies of itself to other computers. Can merely replicate, or can destroy.
  • Trojan horse: a program that enters a computer disguised as somethig desirable and innocuous. Once in the computer the Trojan performs additional activities either immediately or after a period of dormancy dictated by a specific event.
  • spyware: software that collects information about activities on the computer in which it resides while reporting the information back to the operator.
  • phishing: a technique in which a perpetrator sends emails posing as an authoritative body and requests personal infromation from the target.
  • DoS: "Denial of Service" is an atack in which the computer is overloaded with messages. In order to achieve this, the attacker plants software on numerous computers that will generate messages when a signal is given.
Protection
  • firewall: a chokepoint in a network which filters messages passing through it.
  • spam filter: firewalls designed to block unwanted emails.
  • proxy server: a software unit that acts as an intermediary between a client and a server with the goal of shielding the client from adverse actions of the server.
  • antivirus software: software used to detect and excise known viruses and other infections.
  • cryptography: the study of sending and receiving messages securely in the presence of adversaries.
  • public key encryption: a technique in which the encryption and decryption algorithms are known as is one of the keys but the encoded message remains secure, the private key is required to decrypt the message but is held at the intended destination.
