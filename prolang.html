Term Definition
Machine language The lowest-level programming language understood by computers, consisting of binary code that can be executed directly by the CPU.
Debugging The process of finding and fixing errors or defects in a computer program.
Assembler A program that converts assembly language code into machine language code.
Assembly language A low-level programming language that uses mnemonics to represent machine language instructions.
Translator A program that converts code written in one programming language into code written in another programming language.
Compiler A program that translates high-level programming language code into machine language code.
Interpreter A program that executes high-level programming language code directly, without the need for compilation.
Cross-platform software Software that can run on multiple operating systems or hardware platforms.
Natural language A human language, such as English or Spanish, used for communication between people.
Formal language A language with a precise, formal grammar used for communication between computers or between humans and computers.
Machine independence The ability of software to run on different hardware platforms without modification.
High-level programming language A programming language that uses natural language elements and abstraction to make it easier to write and read code.
Object-oriented programming (OOP) A programming paradigm that uses objects to represent data and the operations that can be performed on that data.
Scripting language A programming language used for scripting or automating tasks, such as shell scripting or web scripting.
Data types:
Integer A data type used to represent whole numbers, such as -3, 0, and 42.
Float A data type used to represent decimal numbers, such as 3.14.
String A data type used to represent text or other types of character data, such as "hello world".
Primitive data type A basic data type that is not composed of other data types, such as integers, floats, and strings.
Data structure A way of organizing and storing data in a computer program, such as arrays, linked lists, and trees.
Python collection data structures
List A collection of items that can be of different data types and is ordered and changeable. It is represented in Python using square brackets [ ].
Tuple A collection of items that can be of different data types and is ordered and unchangeable. It is represented in Python using parentheses ( ).
Set A collection of unique items that is unordered and changeable. It is represented in Python using curly braces { }.
Dictionary A collection of key-value pairs that is unordered and changeable. It is represented in Python using curly braces { }, with each key-value pair separated by a colon :.
Function A block of code that performs a specific task and can be reused multiple times in a program.
Local variable A variable that is declared and used only within a specific block or function.
Global variable A variable that is declared outside of any function or block and can be accessed from anywhere in the program.
