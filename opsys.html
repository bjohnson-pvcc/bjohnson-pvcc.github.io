Operating Systems Terminology
TERM DEFINITION
operating system The software that controls the overall operation of a computer.
job queue A storage organization in which objects (jobs) are ordered in a FIFO fashion to wait their turn to be executed
interactive processing A feature that allows the program being executed to carry on dialogue with the user
time-sharing A feature of operating systems that allows them to provide servixe to multiple users simultaneously
multiprogramming One means of implementing time-sharing, in which time is divided into intervals and then the
execution of each job is temporarily set aside and another is allowed to execute during the next interval at a time
application software Consists of the programs for performing tasks particular to the machine's utilization
system software Performs those tasks that are common to computer systems in general. It provides the infrastructure that
the application software requires.
utility software A class of system software other than the operating system that consists of software units. The majority of utility
consists of programs for performing activities that are fundamental to ocmputer installations but not included in the OS
graphical user interface A user interface in which objects are presented pictorially on the display as icons.
shell A user interface that communicates via textual messages using a keyboard and a monitor screen.
kernel The internal part of an operating system containing the sofware compnents that perform very basic functions required by the computer installation
device driver Another component of the kernel which are the software units that communicate with the controllers to carry out operations on the peripheral devices attached to the machine.
file manager A software component of the kernel whose job is to coordinate the use of the machine's mass storage facilities.
memory manager A software component of the kernel who job is to coordinate the machine's use of main memory.
virtual memory Illusory memory created by technique called paging in which the memory manager rotates programs and data back and forth between main memory and mass storage
boot strapping A procedure performed by the computer each time it is turned on in which the OS is transferred from mass storage into main memory.
ROM memory that can only be read but not altered.
deadlock A condition in which two or more processes are blocked from progressing because each is waiting for a resource that is allocated to another.
spooling A technique in which data is held for otuput at a later but more convenient time.
super user (or administrator) The person who is able to establish accounts, they are identified by name and password. The super user can alter settings withing the OS, modify packages, and adjust the privileges of other users.
multi-factor authentication A system in which the user must provide the password in addition to a special access code provided to the user on another device.
