All About GitHub

  • Q. What is a version control system?
  • A. A system used for managing and tracking changes made to software code.

  • Q. What is Git?
  • A. A free and open source distributed version control system.

  • Q. What is Github?
  • A. A for-profit company that offers a cloud-based Git repository.

  • Q. What is the difference between Git and Github?
  • A. Git is a version control system while GitHub is a cloud based hosting service which allows you to manage Git repositories.

  • Q. Who started Github and how was it started?
  • A. Tom Preston-Werner, Chris Wanstrath, P. J. Hyett and Scott Chacon. Github started as a bootstrapped start-up business.

  • Q. What company owns it now?
  • A. Microsoft

  • Q. How much does a Github account cost?
  • A. It is free for the "Free" plan, $48 for the "Team" plan and $231 for the "Enterprise" plan.

  • Q. What is the Octocat?
  • A. Octocat is the GitHub's mascot, an anthropomorphized cat with five arms.
